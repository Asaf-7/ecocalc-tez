# EcoCalc (Karbon Ayak İzi & ROI Skorkartı)

Tek sayfalık Vite + React + TypeScript + Tailwind CSS uygulaması.

## Kurulum
1. Node.js (LTS) yükle.
2. Proje klasöründe:
   - `npm install`
   - `npm run dev`

## Kullanım
Soldan aylık tüketim ve birim fiyatları gir.
Sağdaki panoda karbon toplamı, emisyon dağılımı (PieChart) ve ROI amortisman süresi anlık güncellenir.

## Not
ROI hesabında GES yıllık üretim varsayımı: `1 kWp ~ 1200 kWh/yıl`.
